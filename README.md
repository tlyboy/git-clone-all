# git-clone-all

Git clone all repositories.

## Install

```bash
npm install -g git-clone-all
```

## Usage

```text
Usage: git-clone-all [options] <user>

Git clone all repositories

Options:
  -V, --version  output the version number
  -h, --help     display help for command
```
